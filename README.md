# 👗 Ropero API

Catalogue of all the clothes you own, and how to take care of them.

> Made for [Ropero](https://github.com/brandonporcel/ropero).

## 🏢 Install

```bash
git clone https://github.com/brandonporcel/ropero-backend.git
cd ropero-backend
npm i
npm run dev
```
